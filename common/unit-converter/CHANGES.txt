수정 반영 내용

1. 작은 값 표시 오류 수정
- 기존: 소수점 6자리 고정 반올림 때문에 1 eV → J 같은 값이 0으로 표시됨
- 수정: 1e-6보다 작은 값과 1e9 이상 값은 과학적 표기법으로 표시

2. 단위 계수/이름 정리
- mmHg/Torr: 1 atm = 760 mmHg = 760 Torr 기준으로 통일
- Mach: 15℃ 공기 기준 근삿값임을 이름에 표시
- year: 365일 기준임을 표시
- gal/fl oz: 미국 단위임을 표시
- hp: mechanical hp로 표시
- cal: 열화학 cal로 표시

3. 고등학교 과학용 단위 추가
- 길이: µm, nm, pm
- 넓이: mm²
- 부피: µL, nL, cm³, dm³
- 질량: µg, ng
- 시간: ms, µs, ns
- 기본 변환: 밀도 범주 추가
- 물리: 전하량/전류/전압/저항을 분리, 파장 범주 추가
- 압력: hPa, Torr 추가
- 에너지: Wh 추가
- 화학: nM, mol/m³, mg/mL, g/mL, mg/dL, hPa, bar 추가
- 생명과학: cm³ 추가
- 지구과학: kPa, mmHg 추가

4. HTML/CSS 조정
- 결과 입력창을 text 타입으로 변경하여 1.602177e-19 같은 과학적 표기법 표시 안정화
- 입력값 step="any" 추가
- 과학적 표기법 안내 문구 추가
- 긴 단위명에 맞게 선택창 폭 확대
